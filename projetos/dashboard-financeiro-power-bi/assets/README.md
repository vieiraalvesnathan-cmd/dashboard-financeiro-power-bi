# Assets

Use esta pasta para armazenar imagens demonstrativas do dashboard com dados ficticios.

Sugestoes de arquivos:

- `visao-geral.png`
- `despesas.png`
- `saldo.png`
- `metas.png`
- `patrimonio.png`
- `planejamento-futuro.png`

Antes de publicar qualquer imagem, confirme que:

- todos os dados sao ficticios;
- nao aparecem nomes pessoais;
- nao aparecem valores reais;
- nao aparecem instituicoes, contas ou informacoes privadas.
